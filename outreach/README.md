# Outreach README

This folder is the default sink for Hermes outreach sync.

## Folder purpose
- Leads/: unqualified prospects (need research)
- Partners/: active partners/publishers (one note per partner)
- Emails/: drafts + important threads (link to partner notes)

## Minimum fields for a Partner note
- Contact(s)
- Program / network
- Status (Prospecting / Contacted / Negotiating / Live / Paused)
- Last touch date
- Next action
