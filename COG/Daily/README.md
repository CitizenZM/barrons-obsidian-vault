# Daily Learning (COG)

This folder is the default sink for Hermes learning sync.

## File naming
- One note per day: YYYY-MM-DD.md

## Recommended structure per daily note
1) Sources (links)
2) Key takeaways (3-7 bullets)
3) Decisions / Next actions (actionable)
4) Tags

## Tags
Use a small stable set, e.g.
- #learning
- #affiliate
- #fraud
- #growth
- #product
- #ai-search
